# Obsidian Knowledge Layer

This directory contains the Obsidian vault documentation and templates used by the AI Workbench for audit‑able knowledge capture.

## Vault location

The operational vault lives in `docs/obsidian/vault/`. Open this directory as a vault in Obsidian to view and edit the knowledge base.

## Templates

Base markdown templates are stored in `docs/obsidian/templates/`. The vault contains a copy of these templates in `docs/obsidian/vault/_Templates/` for use within the vault.
